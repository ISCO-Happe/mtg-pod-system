# MTG Pod Randomizer - Installation & Setup Guide

## Quick Setup Instructions

### Windows Desktop
```powershell
# Navigate to project directory
cd C:\Users\chappe\Desktop\mtg_pod_system

# Install Python dependencies
py -m pip install rich

# Run the application
py main.py
```

### Mobile Setup (Android/iOS)
See README.md for detailed mobile setup instructions

## Requirements

- Python 3.7+
- Rich library (for terminal interface)
- Modern web browser (for web interface)
- Terminal app for mobile use

## First Time Usage

1. Add players using "Manage Players" option
2. Configure pod sizes in "Settings" 
3. Create pods with "Create Random Pods"
4. Save results to history if needed

Enjoy your MTG gaming sessions!
